# AI Chatbot

An intelligent conversational AI chatbot built using the **Qwen/Qwen2-1.5B-Instruct Large Language Model (LLM)** and **Flask**.  
The chatbot understands user queries and generates human-like responses.  
It also integrates **T5-Small** for text summarization.
